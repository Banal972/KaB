KaB

※ KaB는 잇다소프트에서 사용되는 프로젝트에 원할하게 사용됩니다.

기본 사용법


    <script src="./Kab.js"></script>
    <script>
        kab.start({
            duration : 1, // 기본값은 1초 입니다. kab-duration 로 조절 할수있지마 전체 조절을 의미합니다.
            marker : true, // 마커가 어디서 찍히는지 파악할수있게 됩니다.
        });
    </script>

Fade

일반적인 Fade in-out 효과

Fade-up

<div kab-fade="up">

Fade-bottom

<div kab-fade="bottom">

Fade-left

<div kab-fade="left">

Fade-right

<div kab-fade="right">

Utils

kab 를 하용하기 간편한 유틸 사용법

Delay

애니메이션 딜레이를 줄수있습니다. 예를 들어 1은 1초 3은 3초 입니다.

<div kab-fade="up" kab-delay="3" >

Start

애니메이션 시작 위치를 설정 할수있습니다. .클래스이름 을 넣어주면 그 클래스를 기준으로 애니메이션이 실행 됩니다.

<div kab-fade="up" kab-start=".startTest" >

Duration

애니메이션 속도를 조절할수 있습니다.

<div kab-fade="up" kab-duration="1.6">
